# 🎯 Number Guessing Game

A feature-rich Number Guessing Game built using Python, Tkinter, and Pygame.

Version 2.0 introduces a graphical user interface, sound effects, score tracking, high-score saving, and multiple difficulty levels.

---

## Features

* 🎮 Interactive Tkinter GUI
* 🎯 Random number generation
* 🟢 Easy, 🟡 Medium, and 🔴 Hard modes
* 📈 Live score tracking
* 🏆 Persistent best score system
* 🔊 Sound effects
* 💡 Too High / Too Low hints
* 🔄 New Game option
* 📋 Built-in game rules
* 🖼️ Custom icons and styling

---

## Difficulty Levels

| Difficulty | Range   | Attempts |
| ---------- | ------- | -------- |
| Easy       | 1 - 50  | 10       |
| Medium     | 1 - 100 | 7        |
| Hard       | 1 - 500 | 5        |

---

## How To Play

1. Select a difficulty level.
2. Enter a number in the input box.
3. Click the **GUESS** button.
4. Follow the hints:

   * Too High
   * Too Low
   * You Win
5. Guess the correct number before running out of attempts.
6. Earn points based on difficulty and remaining attempts.
7. Try to beat your best score.
